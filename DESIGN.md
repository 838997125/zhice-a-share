# 智策设计规范 (Taste-Skill v2 校准)
> 本文档遵循 Taste-Skill 反Slop设计准则，所有前端修改必须遵循本规范
> 最后更新: 2026-07-09

## 品牌定位
- **产品**: 智策 · A股多智能体分析终端
- **气质**: 专业金融科技、克制、可靠、高信噪比
- **受众**: 投资者、交易员、量化分析师
- **情绪关键词**: 冷静、可信、高效、精准
- **主题**: 暗色模式优先（专业交易软件惯例）

---

## §1 色彩系统 (金融行业校准)
### 核心主色
| 颜色 | 色值 | 用途 |
|------|------|------|
| 品牌金 | `#F0B90B` (Binance同款交易黄金) | CTA按钮、进度完成、激活状态、关键指标高亮 |
| 品牌金软背景 | `rgba(240,185,11,0.1)` | 标签背景、激活tab背景、focus环 |
| 上涨红 (A股惯例) | `#EF4444` | 买入信号、上涨、看涨 |
| 下跌绿 (A股惯例) | `#22C55E` | 卖出信号、下跌、看跌 |
| 中性蓝 | `#3B82F6` | 链接、次要信息 |

### 暗色模式中性色阶
| 层级 | 色值 | 用途 |
|------|------|------|
| 页面背景 | `#0A0E1A` | 全局底色 |
| 卡片背景 | `#121929` | 一级卡片、输入框 |
| 卡片悬停 | `#172035` | hover状态 |
| 提升背景 | `#1A2338` | 卡片头部、下拉弹出层 |
| 边框 | `rgba(42,52,84,0.7)` | 所有分割线、边框，禁用纯不透边框 |
| 主文字 | `#E2E8F0` | 标题、正文 |
| 次级文字 | `#94A3B8` | 正文说明、描述文字 |
| 弱化文字 | `#5A6882` | 版权、版本、时间戳 |

### 色彩纪律
1. ❌ 禁止在大面积背景使用渐变（唯一例外：品牌logo方块内的渐变）
2. ❌ 禁止高饱和紫色、粉色等非金融行业色彩
3. ✅ 功能色必须搭配低饱和同色背景，不直接用纯色硬撞
4. ✅ 所有交互元素必须有状态色对应：默认/hover/active/disabled

## §2 排版
### 字体栈
```css
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif;
```
- 代码/数字: `'SF Mono', 'Cascadia Code', 'Fira Code', 'Consolas', monospace`
- 必须开启字体抗锯齿: `-webkit-font-smoothing: antialiased;`

### 字号层级
| 用途 | 字号 | 字重 |
|------|------|------|
| 页面标题 | 1.5rem (24px) | 700 |
| 卡片标题 | 0.9rem (14.4px) | 600 |
| 决策面板标题 | 1.15rem | 700 |
| 正文 | 0.82-0.88rem | 400 |
| 辅助文字/标签 | 0.68-0.75rem | 500 |
| 大数字(指标) | 1.4rem | 700 (mono) |
| 按钮 | 0.9rem | 600 |

### 排版纪律
- 字间距: 标题 `-0.01~-0.02em`，大写标签 `0.05~0.08em`
- 行高: 正文 1.7~1.75，标题 1.3
- ❌ 禁止全大写中文，英文大写标签必须加字间距
- ✅ 数字/代码必须用等宽字体对齐

## §3 空间与圆角
### 间距系统 (4px基准)
| 场景 | 间距 |
|------|------|
| 页面外边距(桌面) | 32px 40px 80px |
| 页面外边距(移动) | 20px 16px 60px |
| 卡片内边距 | 16px 18px |
| 卡片网格gap | 20px(桌面) / 16px(移动) |
| 元素小间距 | 4px/6px/8px |
| 元素中间距 | 10px/12px/14px |
| 元素大间距 | 16px/20px/24px |

### 圆角层级（严格递进，禁止一刀切）
| 元素 | 圆角 |
|------|------|
| 小标签、徽章 | 20px (胶囊) |
| 极小元素(输入框、按钮) | 8px |
| 普通卡片、弹出层 | 10px |
| 大卡片、决策面板 | 14px |
| 品牌logo方块 | 10px |

### 空间纪律
- ❌ 禁止所有元素使用统一6px/10px圆角
- ❌ 禁止均匀等距布局，要有疏密节奏
- ✅ 卡片之间、区块之间留白要大于内部留白
- ✅ 移动端点按区域最小44x44px

## §4 阴影与质感
### 阴影层级
| 场景 | 阴影值 |
|------|--------|
| 输入框、小按钮 | `0 1px 2px 0 rgba(0,0,0,0.3)` |
| 卡片、弹出层 | `0 8px 24px -8px rgba(0,0,0,0.4), 0 3px 8px -3px rgba(0,0,0,0.2)` |
| 主CTA按钮hover | `0 6px 20px -4px rgba(240,185,11,0.25)` |
| 激活状态卡片 | `0 0 0 1px rgba(240,185,11,0.15), 0 8px 24px -8px rgba(0,0,0,0.4)` |

### 质感纪律
- ❌ 禁止滥用毛玻璃效果，仅顶层下拉浮层可使用轻量blur
- ❌ 禁止无阴影纯边框纸片设计
- ✅ 暗色模式下阴影透明度要低于浅色模式
- ✅ 阴影方向统一向下偏，模拟真实光照

## §5 动效规范
### 缓动函数（统一使用，禁止linear）
- 元素入场: `cubic-bezier(0.22, 1, 0.36, 1)` (快出慢入，自然回弹)
- 标准交互(hover/focus): `cubic-bezier(0.5, 0, 0.5, 1)`

### 时长规范
| 场景 | 时长 |
|------|------|
| 微交互(hover、focus) | 0.2s |
| 页面整体入场 | 0.7s (延迟0.05s) |
| 面板入场(决策面板) | 0.5s |
| 状态过渡(边框、颜色) | 0.2-0.3s |
| 呼吸动画(pulse) | 2s 循环 |
| Spinner旋转周期 | 1.2s |

### 动效细节
- 页面入场: 从 `translateY(8px) + opacity:0` 渐入，禁止纯fadeIn
- 按钮hover: 上移1px + 阴影扩散，按下回到原位
- 卡片激活: 轻微上移2px + 金色外发光
- Tab切换: 底部指示条宽度从0增长，而非直接变色
- ❌ 禁止弹跳、浮夸抖动、炫技动效
- ✅ 必须遵循 `prefers-reduced-motion` 关闭动画

## §6 组件特殊规则
### Progress River (进度河流-签名组件)
- 节点大小38px，内边距舒适，不要太挤
- 连接线2px粗，完成态变金色
- 当前节点呼吸动画扩散范围6px，不要太大
- 标签文字要换行显示完整，不要截断

### Agent Cards (分析师卡片)
- 激活态金色边框+阴影+轻微上浮
- 完成态降低不透明度到0.9但不要太灰
- 多agent标签页要有底部金色指示条
- 卡片内容区滚动条6px粗，hover加深，不要4px过细

### 决策面板
- 信号标签要有对应颜色阴影，不要平面色块
- 指标区和文字说明区用竖线分割，不要混在一起
- 出现时有从下往上的入场动画

## §7 预检清单(所有修改必须通过)
1. ✅ 没有无意义渐变（仅logo允许）
2. ✅ 圆角按层级使用，没有统一6px/8px一刀切
3. ✅ 所有动效使用贝塞尔曲线，没有linear动画（除了循环旋转）
4. ✅ A股 红涨绿跌 色彩规则正确，没有搞反
5. ✅ 文字对比度符合WCAG AA标准（暗色下≥4.5:1）
6. ✅ 移动端做了断点适配，横滚不溢出
7. ✅ 阴影有层次，不是全页面统一大阴影或无阴影
8. ✅ 交互元素都有hover/active/focus状态
9. ✅ 所有数字/代码使用等宽字体
10. ✅ 没有emoji作为功能图标（搜索、箭头等必须用SVG）
