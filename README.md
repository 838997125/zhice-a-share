# 智策 · A股多智能体分析终端

> 基于 LLM 多智能体协作的 A 股投资决策分析平台

## 简介

智策是一个基于多智能体大语言模型的 A 股交易决策分析系统。系统派遣 10 个 AI 分析师，分别从市场技术面、社交媒体情绪、新闻舆情、基本面四个维度对目标股票进行深度分析，经过牛熊辩论和多层风险评估后，输出最终交易建议。

### 核心特性

- **多智能体协作**：10 个专业 AI 分析师分工合作，涵盖技术面、情绪面、新闻面、基本面
- **牛熊辩论机制**：多空双方多轮辩论 + 裁判裁决，避免单一视角偏见
- **三层风险评估**：激进 / 保守 / 中立三方独立讨论 + 风险裁判终裁
- **实时流式输出**：基于 SSE 的实时分析过程推送，每个智能体的思考过程可见
- **A 股原生适配**：使用 akshare 数据源，支持沪深京全市场，红涨绿跌
- **多模型支持**：支持火山方舟、DeepSeek、GPT、Claude、Gemini、通义千问等 12+ 预设模型
- **中文决策输出**：最终决策依据（rationale）直接输出中文

## 快速开始

### 环境要求

- Python 3.12+
- [akshare](https://github.com/akfamily/akshare)（`pip install akshare`）
- FastAPI + Uvicorn（`pip install fastapi uvicorn`）
- 多智能体分析框架（`pip install tradingagents`）

### 配置

1. 复制环境变量模板：
```bash
cp .env.example .env
```

2. 编辑 `.env`，填入你的 LLM API Key 和端点：
```bash
# 示例：使用火山方舟 ARK
OPENAI_API_KEY=your-api-key-here
OPENAI_API_BASE=https://ark.cn-beijing.volces.com/api/coding/v3
```

3. 也可以使用其他 OpenAI 兼容接口（DeepSeek、通义千问、GPT 等），在 Web UI 的「模型配置」面板中添加。

### 启动

```bash
# 启动 Web 服务（默认端口 8460）
bash start_web.sh

# 或指定端口
bash start_web.sh 9000
```

访问 `http://localhost:8460` 即可使用。

### CLI 方式

```bash
# 分析贵州茅台（默认）
python run_analysis.py

# 指定股票和日期
python run_analysis.py 000001 2025-06-16    # 平安银行
python run_analysis.py 300750 2025-06-16    # 宁德时代
```

| 2 | 📱 社交媒体分析师 | 社交媒体情绪分析 | quick |
| 3 | 📰 新闻分析师 | 新闻情绪分析 | quick |

| 5 | 📝 情况总结器 | 汇总分析师报告 | quick |
| 6 | ⚖️ 牛熊研究员 | 基于报告进行多轮辩论 | quick |
| 7 | 👨‍⚖️ 辩论裁判 | 裁决牛熊辩论 | deep |
| 8 | 💰 交易员 | 综合信息做出交易决策 | quick |
| 9 | 🛡️ 风险辩论 | 激进/保守/中立三方讨论 | quick |
| 10 | ⚡ 风险裁判 | 最终风险评估 | deep |
| 11 | ✅ 完成 | 输出最终决策 | - |

## 模型配置说明

框架支持两个模型层级：

- **deep_think_llm**：辩论裁判 + 风险裁判（复杂推理场景）
- **quick_think_llm**：其他所有角色（快速响应场景）

Web UI 模型管理面板支持：
- 添加 / 删除 / 测试模型连通性
- `models.json` 持久化配置
- 角色级模型覆盖（映射到 deep/quick 两个层级）
- 12+ 预设模型模板
